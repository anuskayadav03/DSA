## 🚀 How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/anuskayadav03
   ```

2. Navigate to the project folder:

   ```bash
   cd javafolder
   ```

3. Compile and run any Java file:

   ```bash
   javac FileName.java
   java FileName
   ```
